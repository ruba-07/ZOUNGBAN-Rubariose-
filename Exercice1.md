Question 1. Définissez dans vos propres mots les termes suivants : état, espace d’états, action,
fonction de succession, graphe d’états.

Réponse :

État :
Une situation précise du système à un instant donné. C’est une photographie du monde à un moment T, contenant toutes les informations nécessaires pour prendre une décision.

Espace d’états :
L’ensemble de tous les états possibles que le système peut avoir. C’est comme une carte qui répertorie toutes les configurations imaginables du problème.

Action :
Une opération qui permet de passer d’un état à un autre. Une action modifie la situation actuelle .

Fonction de succession :
C’est ce qui indique quels états on peut atteindre depuis un état donné en appliquant une action.


Graphe d’états :
Une représentation sous forme de graphe où les nœuds sont les états,les arêtes sont les actions reliant un état à un autre
C’est une carte graphique du problème.


Question 2. Quelle est la différence entre un état du monde réel et une description d’état ? Pourquoi
cette distinction est-elle utile ?

Réponse :

Un état du monde réel est la vraie situation physique telle qu’elle existe dans la réalité : tous les détails, même ceux que nous ne percevons pas.

Une description d’état est une version simplifiée ou abstraite de cet état, utilisée par l’agent ou l’algorithme pour prendre une décision.
Elle contient seulement l’information utile.

Utilité :

Parce qu’on n’a pas besoin de tous les détails réels pour résoudre un problème.

Cela permet de réduire la complexité, de modéliser le monde plus facilement.
